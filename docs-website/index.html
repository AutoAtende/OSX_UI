OSX_UI

macOS-style desktop window manager framework for web apps

Give your single-page application the look and feel of a macOS desktop, complete with draggable/resizable windows, a menu bar, a dock, theming, and a built-in client-side router.

My App

Window Manager

Draggable, resizable windows with minimize, maximize, close, and z-order focus.

Menu Bar

macOS-style menu bar with dropdowns, keyboard shortcuts, and dynamic items.

20+ Components

Buttons, forms, tables, tabs, accordions, toasts, and more with macOS styling.

Dark & Light Themes

Built-in themes with easy customization via CSS custom properties.

Client-side Router

History API routing with automatic link interception for SPA navigation.

Responsive

Windows become fullscreen on mobile, with drag/resize disabled.

Quick Install

npm install osx-ui

Quick Start

import { createApp } from 'osx-ui';

const app = createApp({
  container: document.getElementById('root'),
  appName: 'My App',
  theme: 'dark',
  showClock: true,
});

app.router.start();
