Quick Start

Get up and running with OSX_UI in minutes

Installation

Install OSX_UI via npm:

npm install osx-ui

Import CSS

Add the CSS to your HTML file:

<link rel="stylesheet" href="node_modules/osx-ui/css/osx-ui.css" />

Or import directly in JavaScript:

import 'osx-ui/css';

Create Your App

Use the createApp function to initialize the desktop environment:

import { createApp } from 'osx-ui';

const app = createApp({
  container: document.getElementById('root'),
  appName: 'My App',
  theme: 'dark',
  showClock: true,
  menus: [
    {
      id: 'file',
      label: 'File',
      items: [
        { label: 'New Window', action: () => openWindow() },
        { type: 'separator' },
        { label: 'Quit', action: () => app.destroy() },
      ],
    },
  ],
  routes: [
    { pattern: /^\/$/, handler: () => showHome() },
  ],
});

app.router.start();

function showHome() {
  const pos = app.centerPos(600, 400);
  app.wm.createWindow({
    id: 'home',
    title: 'Welcome',
    content: '<h1>Hello, World!</h1>',
    ...pos,
    width: 600,
    height: 400,
  });
}

API Overview

The createApp function returns an app object with the following properties:

Property Description
app.wm WindowManager for creating/managing windows
app.menuBar MenuBar with your menus rendered
app.router Router with your routes registered
app.theme ThemeManager for switching themes
app.desktop The desktop DOM element
app.centerPos(w, h) Helper to center a window on the desktop
app.destroy() Tears down everything
